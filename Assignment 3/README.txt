Assignment 3: Player and Camera Control

COMP 437: Computer Game Development Spring 2018

Part 1 (30 points): Implement a first-person character and camera setup in Unity.

Part 2 (30 points): Implement a third-person character and camera setup in Unity.

Part 3 (40 points): Add one feature per team member to one or both of the control setups above, or implement a new control or camera setup. Some examples:

• Add clipping capabilities to the third-person camera
• Add camera distance controls to the third-person camera
• Implement a side-view camera
• Obstruct player movement on steep terrain, or cause the player to slide downhill on steep terrain.
• Let the player jump, or double jump, or fly.
